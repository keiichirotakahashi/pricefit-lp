## 課題想起セクション

### 見出し

EC業務でこんなお悩みはありませんか？

### 課題リスト

1. 価格調査・更新にかかる時間が膨大
    - 見出し
        - 価格調査と価格更新に多くの時間を費やしている
    - 本文
        - 競合商品を検索して価格をExcelにまとめ、必要に応じて価格を変更…この繰り返しに多くの時間を取られ、売上を伸ばすための施策に手が回らない。
2. 最適な価格を設定できておらず利益を逼迫
    - 見出し
        - 最適な価格設定ができず利益を最大化できていない
    - 本文
        - 高すぎて売れない、安すぎて利益を削ってしまう。価格の見直しが追いつかず、売上・利益の機会を逃してしまう。
3. 既存のツールの導入ハードル
    - 見出し
        - 既存のツールは複雑かつ高価で導入ハードルが高い
    - 本文
        - 既存のツールは多機能ゆえに設定や運用が煩雑で、料金も高くてなかなか手が出せない。

## ソリューションセクション

### 見出し

PriceFitがまるごと解決

### ソリューションリスト

1. 自動化
    - 見出し
        - 価格の調査と更新を自動化
    - 本文
        - カンタンな設定をするだけで、競合価格の調査から自社価格の更新までを自動で実行。人手に頼らず、常に理想の価格をキープできます。
2. 利益最大化
    - 見出し
        - 最適な価格設定で利益を最大化
    - 本文
        - 最安値に追随するだけでなく、設定した下限・上限や利益率を考慮して価格を調整。売れる価格をキープしながら、しっかり利益も残せます。
3. 低い導入ハードル
    - 見出し
        - シンプルかつおトクで気軽に試せる
    - 本文
        - 必要な機能に絞ったシンプルな操作性で誰でも迷わずに使えます。また業界最安水準の料金のため気軽に試すことができます。

## 利用の流れセクション

### 見出し

ご利用ステップ

### ステップリスト

1. **検索条件を登録**
    - 見出し
        - 商品の検索条件を登録
    - 本文
        - 商品名・JANコード・ASINなどを入力し、価格調査の条件を登録します。
2. 自動で価格を調査
    - 見出し
        - 自動で価格調査を開始
    - 本文
        - 設定条件に基づき、主要モールの価格を自動で収集。最新の競合価格をリストにまとめます。
3. 自動で最適な価格に更新
    - 見出し
        - 自動で最適な価格に更新
    - 本文
        - 競合価格と自社ルールをもとに、自動で価格を更新。高すぎれば値下げ、安すぎれば値上げを行い、最適価格をキープします。

## FAQセクション

### 見出し

よくあるご質問

### Q&Aリスト

| 質問 | 回答 |
| --- | --- |
| 価格調査に対応しているサイトは？ | Amazon、楽天市場、Yahoo!ショッピングに対応しています。順次、他モールや価格比較サイトへの対応も検討しています。 |
| 1日に複数回の調査は可能ですか？ | はい。標準で1日数回の調査に対応しており、オプションで10分ごとなど高頻度の調査にも対応予定です。 |
| ポイントや送料は考慮されますか？ | いいえ。現在は本体価格を取得対象としていますが、将来的に送料やクーポンを含めた「実質価格」での比較機能を追加予定です。 |
| 自動価格更新はできますか？ | はい。設定した下限価格や利益率を守りながら、自社価格を自動で更新できます。自社同士の出品での「値下げ合戦」を避ける仕組みも備えています。 |
| 利用料金はいくらですか？ | ベータ版期間は先着20社限定で無料でご利用いただけます。20社を超える場合や正式リリース後も、業界最安水準での提供を予定しています。 |
| 無料トライアルはありますか？ | はい。現在は先着20社限定でベータ版を無料提供しています。ベータ版は期間制限なく無料でご利用いただけます。20社を超える場合や正式リリース後は、2週間程度の無料トライアルをご用意する予定です。 |
| 最低利用期間はありますか？ | いいえ。最低利用期間の縛りはなく、1ヶ月単位でのご契約となります。 |
